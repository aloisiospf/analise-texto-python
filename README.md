# Análise de Texto com Python

Projeto introdutório para classificação simples de textos em positivo, negativo ou neutro.

## Funcionalidades
- Leitura de texto pelo terminal
- Classificação baseada em palavras-chave
- Aplicação de lógica básica para análise textual

## Tecnologias
- Python

## Como executar

```bash
python main.py
```

## Descrição
Este projeto foi desenvolvido para praticar lógica de programação, manipulação de strings e introdução à análise de dados textuais com Python.
